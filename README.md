#  DigiTools 

> A high-performance, modern e-commerce platform for digital assets and productivity tools. Built with **React**, **Vite**, and **Tailwind CSS**.

🔗 **Live Demo:** [unrivaled-pothos-1cea7a.netlify.app](https://unrivaled-pothos-1cea7a.netlify.app/)

-----

## 🧾 Project Overview

**DigiTools** is a sleek, responsive digital storefront designed for selling software, design templates, and workflow assets. It features a streamlined shopping experience, including a dynamic product grid, tabbed category filtering (Products/Cart), and a multi-tier pricing structure.

## ✨ Key Features

  * **⚡ High-Performance Storefront** — Fast-loading UI built with Vite and React.
  * **🛒 Dynamic Shopping Cart** — Seamlessly add/remove digital products with real-time total calculation.
  * **🏷️ Product Filtering** — Toggle between catalogs and your active cart using a tabbed navigation system.
  * **💎 Premium Product Cards** — Detailed product displays with pricing, features, and category tags (e.g., "AI Writing", "Design").
  * **💳 Multi-Tier Pricing** — Dedicated section for Starter, Pro, and Enterprise subscription plans.
  * **📱 Fully Responsive** — Tailored experience for desktop, tablet, and mobile users via Tailwind CSS.
  * **🔔 Interactive Feedback** — Integrated toast notifications for cart actions.

## 🚀 Getting Started

### Prerequisites

  * Node.js (v18.0 or higher)
  * npm or yarn

### Installation & Local Setup

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/tanvirislam06408/digi-tools.git
    ```

2.  **Navigate to the project folder:**

    ```bash
    cd digi-tools
    ```

3.  **Install dependencies:**

    ```bash
    npm install
    ```

4.  **Run the development server:**

    ```bash
    npm run dev
    ```

## 📂 Project Structure (Highlights)

  * `src/components/` — Reusable UI components (Navbar, Pricing, ProductCard).
  * `src/context/` — Global state management for the shopping cart.
  * `src/assets/` — Images and branding assets for digital tools.
